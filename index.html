📋 Règlement du tournoi interclub
🎯 Format interclub

Le tournoi interclub de pickleball oppose des équipes de 2 joueurs fixes. Les deux partenaires restent ensemble pour toute la durée du tournoi — il n'y a pas de rotation des partenaires comme dans le format Americano.

  • Chaque équipe est représentée par un nom et un club d'appartenance.
  • Les deux joueurs d'une équipe ne changent jamais de partenaire.
  • Le tournoi se déroule en deux sessions : matin et après-midi.
  • Si le tournoi utilise 2 salles, chaque équipe reste dans la même salle toute la journée.
📅 Organisation des sessions

Matin et après-midi

  • Le tournoi prévoit 3 ou 4 matchs le matin et le nombre complémentaire l'après-midi (4 ou 3), pour un total de 7 matchs par équipe.
  • Si une coupure déjeuner est configurée, les équipes ont une pause définie entre les deux sessions.
  • Les terrains sont remplis au maximum à chaque tour pour limiter les temps d'attente.

Salles

  • En configuration 1 salle : toutes les équipes jouent dans le même gymnase.
  • En configuration 2 salles : les équipes sont réparties entre les deux salles dès le début du tournoi et n'en changent pas. L'admin peut affecter manuellement chaque équipe à une salle, ou laisser la répartition automatique.
⚖️ Appariement des équipes

À chaque tour, l'algorithme génère les matchs en respectant les priorités suivantes :

  • Éviter les équipes du même club : autant que possible, deux équipes provenant du même club ne s'affrontent pas.
  • Minimiser les répétitions : deux équipes qui se sont déjà rencontrées ne rejouent pas ensemble si d'autres options existent.
  • Équilibre de niveau (si activé) : les équipes de niveau similaire sont prioritairement opposées.
  • Si le nombre d'équipes est impair dans une salle, une équipe est exemptée (au repos) à chaque tour, à rotation équitable.
🏆 Points et classement

Points par match

  • Victoire : 3 points
  • Match nul : 2 points pour chaque équipe
  • Défaite : 0 point

Exemption (tour au repos)

  • Une équipe exemptée reçoit 2 points de compensation (équivalent match nul 0–0), garantissant l'équité.
  • La rotation des exemptions est gérée pour que chaque équipe soit exemptée le moins souvent possible et de manière équitable.

Départage (à égalité de points)

  1. Points totaux
  2. Goal-average général (buts marqués − buts encaissés)
  3. Goal-average particulier (confrontation directe entre ex-æquo)
  4. Buts marqués (total)
  5. Nombre de victoires
  6. Nombre de matchs joués
🛡️ Règles pratiques

Avant le tournoi

  • Les équipes s'échauffent au minimum 10 minutes avant leurs premiers matchs.
  • Chaque équipe est responsable de son équipement (raquettes, tenue).
  • Les joueurs doivent être présents sur leur terrain 2 minutes avant l'heure prévue.

Pendant le tournoi

  • En cas d'absence, un match peut être déclaré forfait après 5 minutes d'attente.
  • Toute blessure doit être signalée immédiatement à l'organisateur.
  • En cas de balle étrangère sur le terrain, annoncer « Let ! » pour rejouer le point.

Scores

  • Les scores sont saisis par les organisateurs (rôles admin ou scorer).
  • Le classement se met à jour en temps réel sur tous les appareils connectés.
  • Les spectateurs peuvent suivre le Live et le Classement en accès libre (si le live est ouvert).
